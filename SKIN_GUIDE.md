# 스킨 제작 가이드 (v4.0)

## 개요

사진만 올리면 끝. 3가지를 각각 설정:

| 영역 | 설명 | 권장 이미지 |
|------|------|------------|
| **앱 배경** | 메인 화면 배경 사진 | **1080×2400px** (9:20) |
| **오버레이 이미지** | 플로팅 위젯 배경 | **600×88px** |
| **글씨 색** | 오버레이 텍스트 색상 | HEX 코드 또는 프리셋 12색 |

---

## 1. 앱 배경 사진

메인 앱 화면의 배경.

- **권장**: 1080×2400px (9:20 비율)
- **형식**: PNG, JPG, WebP
- centerCrop 적용 — 어떤 비율이든 깨지지 않음
- 없으면 기본 다크 배경 (#1a1a2e)

---

## 2. 오버레이 이미지

화면 위에 떠 있는 사용량 표시창의 배경.
고정 크기 **300dp × 44dp**에 맞춰 이미지가 표시됨.

- **권장**: 600×88px (2x 기준)
- **형식**: PNG (투명 배경 가능), JPG, WebP
- 텍스트가 위에 렌더링되므로, 가운데 영역은 단색/반투명 권장
- 없으면 기본 다크 그라데이션

### 오버레이 이미지 디자인 가이드

```
┌──────────────────────────────────────────────────────┐
│ ▎ ━━━━━━━━━━━ 장식 상단 라인 ━━━━━━━━━━━ ▐ │
│ ▎        🟢 42% │ 주간 35% │ 💰$0.12         ▐ │  ← 텍스트 영역
│ ▎ ━━━━━━━━━━━ 장식 하단 라인 ━━━━━━━━━━━ ▐ │
└──────────────────────────────────────────────────────┘
        600px (너비) × 88px (높이)
```

- **너비 600px**: FULL 모드 텍스트 + 양쪽 장식 포인트 공간
- **높이 88px**: 텍스트 위아래에 특징적인 선/장식 가능
- 양쪽 끝에 디자인 포인트 (테두리, 각진 모서리, 아이콘 등)
- 텍스트 영역(중앙)은 어둡거나 밝은 단색 배경 권장 (가독성)

### 예시 아이디어

| 스타일 | 설명 |
|--------|------|
| 네온 프레임 | 어두운 배경 + 형광색 테두리/라인 |
| 글래스 | 반투명 흰색/검정 + 블러 느낌의 테두리 |
| 메탈 | 그라데이션 은색/금색 + 리벳 느낌 |
| 게임 UI | HP바 스타일 프레임 |
| 미니멀 | 둥근 모서리 + 얇은 선 |
| 캐릭터 | 캐릭터가 양 옆에 서있는 프레임 (투명 PNG) |

---

## 3. 글씨 색

오버레이 텍스트 색상. 앱에서 직접 선택:
- **프리셋 12색**: 흰색, 검정, 빨강, 민트, 노랑, 보라 등
- **HEX 직접 입력**: #FFFFFF, #FF6B6B 등
- **초기화**: 기본 밝은 회색 (#e0e0e0)

**팁**: 오버레이 이미지가 어두우면 밝은 글씨, 밝으면 어두운 글씨.

---

## 고급: .cskin 파일 (선택)

기존 `.cskin` JSON 파일도 계속 사용 가능 (하위 호환).
세부적인 그라데이션, 테두리, 그림자, clipPath 등 고급 설정이 필요할 때.

```json
{
  "name": "스킨 이름",
  "overlay": {
    "background": { "type": "gradient", "colors": ["#0a0820", "#1a0040"] },
    "text": { "color": "#4DDFFC", "shadow": { "color": "#00FFC0", "radius": 8 } },
    "shape": { "cornerRadius": 4, "border": { "color": "#FF00FF", "width": 1 } }
  },
  "app": {
    "backgroundColor": "#0a0820",
    "textColor": "#4DDFFC",
    "accentColor": "#FF00FF",
    "isDark": true
  }
}
```

```
[앱 설정 → 스킨]

  앱 배경 사진     [미리보기] [사진 선택] [제거]
                   → 1080×2400px 사진 업로드

  오버레이 이미지   [미리보기] [이미지 선택] [제거]
                   → 600×88px 이미지 업로드

  글씨 색          [미리보기] [#HEX] [적용] [초기화]
                   → ●●●●●●●●●●●● 프리셋 12색
```
