# Claude 사용량 위젯

Claude Max/Pro 플랜 사용량을 상단 알림창에서 바로 확인할 수 있는 안드로이드 앱.
클로드 앱 설정 > 사용량 화면과 동일한 정보를 알림창에 상시 표시합니다.

## 표시 정보

- **현재 세션**: 5시간 윈도우 사용량 (%) + 재설정까지 남은 시간
- **주간 한도**: 주간 사용량 (%) + 재설정 시점

## 기능

- 상단 알림창에 사용량 상시 고정
- 알림 펼치면 세션/주간 사용량 상세 표시
- "새로고침" 버튼으로 즉시 업데이트
- 세션/주간 80% 도달 시 별도 알림
- WebView로 claude.ai 직접 로그인 (세션 쿠키 자동 추출)
- 부팅 시 자동 시작
- 갱신 주기 설정 가능 (기본 120초)

## 설치

1. Android Studio에서 `android/` 폴더를 프로젝트로 열기
2. Build > Build APK
3. 생성된 APK를 폰에 설치

## 사용법

1. 앱 열기
2. **"claude.ai 로그인"** 버튼 → 평소처럼 Google/이메일로 로그인
3. 로그인 완료되면 자동으로 사용량 표시
4. **"모니터링 시작"** → 상단 알림창에 고정

## 알림 예시

접었을 때:
```

```

펼쳤을 때:

```
주간 한도
▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░ 51% 사용됨
⏱ 2일 3시간 후 재설정
```

## 데스크톱 위젯 (예정)

추후 데스크톱용 위젯도 추가 예정입니다.

**계획:**
- **방식**: Python + tkinter (또는 Electron) always-on-top 위젯
- **인증**: 브라우저 쿠키에서 sessionKey 자동 추출 (browser_cookie3 라이브러리)
- **표시**: 안드로이드 앱과 동일 — 현재 세션 % + 주간 한도 %
- **위치**: 화면 우상단 작은 플로팅 윈도우, 드래그 이동 가능
- **업데이트**: 설정한 주기마다 claude.ai에서 직접 조회
