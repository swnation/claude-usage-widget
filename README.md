# Claude 사용량 위젯

Claude Max/Pro 플랜 사용량을 상단 알림창, 홈 화면 위젯, 플로팅 오버레이에서 바로 확인할 수 있는 안드로이드 앱.
클로드 앱 설정 > 사용량 화면과 동일한 정보를 실시간으로 표시합니다.

## 표시 정보

- **현재 세션**: 5시간 윈도우 사용량 (%) + 재설정까지 남은 시간
- **주간 한도**: 주간 사용량 (%) + 재설정 시점 (요일 포함)

## 기능

- 상단 알림창에 사용량 상시 고정
- 알림 펼치면 세션/주간 사용량 상세 표시
- 홈 화면 위젯 (세션 + 주간 + 리셋 시간 + 색상 퍼센트)
- 플로팅 오버레이 (다른 앱 위에 세션 % 실시간 표시)
- 알림 탭 시 자동 새로고침
- 세션/주간 80% 도달 시 별도 경고 알림
- WebView로 claude.ai 직접 로그인 (Google/이메일)
- 부팅 시 자동 시작
- 갱신 주기 설정 가능 (기본 120초)
- 앱 내 자동 업데이트 (GitHub Release)
- 첫 실행 시 필요한 권한 일괄 설정

## 설치

1. GitHub Releases에서 최신 APK 다운로드
2. 설치 (출처를 알 수 없는 앱 허용 필요)

## 사용법

1. 앱 열기 → 첫 실행 시 권한 설정 안내
2. **"claude.ai 로그인"** 버튼 → Google 또는 이메일로 로그인
3. 로그인 완료되면 자동으로 사용량 표시
4. **"모니터링 시작"** → 상단 알림창에 고정
5. **"플로팅 오버레이 켜기"** → 다른 앱 위에 세션 % 표시

## 예정 기능

### 스킨 시스템
- 기본 (다크 보라 — 현재)
- 몬치치 (따뜻한 브라운/베이지)
- 봄 (핑크/연두)
- 여름 (하늘/바다)
- 가을 (오렌지/갈색)
- 겨울 (블루/화이트)

### 데스크톱 위젯
- Python + tkinter (또는 Electron) always-on-top 위젯
- 브라우저 쿠키에서 sessionKey 자동 추출
